OpenTregna Privacy Policy

Effective date: 25 June 2026

OpenTregna is an independent transit planning and lightweight contribution app for Algerian cities and wilayas.

Contact: transportalgerien@protonmail.com

Data We Collect

OpenTregna does not require an account for the current closed testing build.

The app may process or store:

  • Selected city or wilaya.
  • Favorite lines, favorite stops, favorite addresses, and recent trips stored locally on the device.
  • Optional current location, only after the user taps a location or map feature.
  • User-submitted contribution content, such as a proposed line, proposed stop, report text, and optional map pinpoint.
  • An anonymous contributor identifier generated locally on the device, used to show "My contributions".
  • Optional anonymous app diagnostics or route feedback if the user accepts analytics.

Location Data

OpenTregna may request approximate or precise location permission for route planning, current-location origin selection, map pin placement, or contribution pinpoints.

The app does not run background location tracking. Location is used only after a user action.

Contribution pinpoints are sent to the OpenTregna server only when the user voluntarily submits a contribution. These coordinates are used for moderation and dataset improvement. Public catalog endpoints do not expose stop coordinates.

Contributions

When users submit contributions, OpenTregna stores the city or wilaya, contribution type, title and optional note, optional pinpoint selected on the map, anonymous contributor identifier, and submission status.

Contributions are reviewed before they are used to update published transit data.

Analytics

The current app build does not include advertising SDKs.

If analytics are enabled by the user, OpenTregna may send anonymous aggregated usage or route feedback events to help improve routing quality and coverage. Analytics do not store exact GPS coordinates or personal account information.

Users can manage analytics preferences inside the app.

Data Sharing

OpenTregna does not sell personal data.

Data may be processed by infrastructure providers used to operate the app and API, including Cloudflare for API hosting and database storage.

Map display uses OpenStreetMap-compatible map data and MapLibre rendering. External map or browser links may be opened only after user interaction.

Data Retention And Deletion

Local favorites, recent trips, and preferences remain on the device until the user clears app data or uninstalls the app.

Submitted contributions may be retained for moderation, audit, and dataset quality. To request deletion of submitted contribution data, contact transportalgerien@protonmail.com and include enough detail to identify the contribution.

Children

OpenTregna is not directed to children. The app is intended for general transit information and testing by users able to evaluate transport information responsibly.

Security

OpenTregna uses HTTPS for production API traffic. Admin moderation tokens are not included in the mobile app.

Changes

This policy may be updated as OpenTregna evolves. Material changes will be reflected by updating this page.
